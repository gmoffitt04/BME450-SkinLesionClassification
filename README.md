#Project Description

This goal of this project is to develop a deep learning model to classify skin lesion images, according to their skin disease diagnosis. I worked on a project similar to this 2 summers ago, utilizing an AI/ML model to improve diagnostic accuracy of rare skin diseases. However, I did not work on the training of the model, so I hope to expand on my understanding. I plan to use the HAM1000 dataset, an open-source dataset that contains over 10,000 images of dermatologic conditions. HAM1000 is the highest rated open-source dataset. The limitations of HAM1000 are the relative limited number of images, but I believe that 10,000 will be ample for this project. There are 7 types of lesions represented by the dataset: Bowen's disease, basal cell carcinoma, benign keratosis-like lesions, dermatofibroma, melanoma, melanocytic nevi, and vascular lesions. 

The project will utilize convolutional neural networks (CNNs) to classify medical images and evaluate model performance across modified architectures. I will begin with a baseline CNN, and then modify the architecture to determine how architectural changes affect performance. Furthermore, multiple different parameters will be used to test the effect of model structure and training configuration affect performance. These parameters include: learning rate, batch size, number of training epochs, and optimizer selection. The final train and test results of these experiments will allow me to compare multiple architectural variations and parameter settings, determining the most effective approach for skin lesion classification. This project will allow me to demonstrate my understanding of the course by applying deep learning techniques to a dataset and problem of my choosing.
